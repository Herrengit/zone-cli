# zone-cli
由于公司框架使用种类繁多，框架官方提供脚手架配置差异巨大和打包速度较慢，需要一套统一的脚手架
## 安装
```
npm i zone-cli -g
cnpm i zone-cli -g
yarn add zone-cli -g
```
## 功能
### 项目初始化
```
zone-cli <projectName>
```
### 帮助
```
zone-cli help
```
## 提供模板
### 1. vue 2
自定义配置
1. 是否使用移动端适配
2. js是否兼容安卓4.4.0版本
3. 是否配置微信sdk
4. 是否使用vux
6. 是否多页面应用

### 2. react 16
自定义配置
1. 是否使用移动端适配
2. js是否兼容安卓4.4.0版本
3. 是否配置微信sdk
4. 是否使用redux
6. 是否多页面应用

### 3. react + electron
自定义配置
1. electron 版本选择

### 4. typscript-env 
快速生成typscript环境，用于做实验

### 5. es6-env
快速生成es6+环境，用于做实验,可用修饰器
